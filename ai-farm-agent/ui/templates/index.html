O SISTEMA

Ensinando computadores a operarem sozinhos.

◈ OBJETIVO PRIMÁRIO

O AI Farm Agent não enxerga seu computador apenas como pixels, mas como um mundo estruturado de janelas, botões e workflows que podem ser controlados. É um sistema multi-agente autônomo que transforma Linguagem Natural em Ações no Mundo Real.

O sistema pensa, planeja, executa e verifica — criando planilhas no Excel, enviando mensagens, organizando arquivos e navegando na web de forma 100% autônoma.

◆ A EQUIPE DE AGENTES ◆

🧠

MAESTRO

Haiku 4.5

O Cérebro. Roteia cada tarefa para o especialista certo, decompõe pedidos complexos e coordena a ordem de execução.

📊

⚡ CASCATA DE INTERAÇÃO (L1 → L2 → L3)

A inovação central. O sistema desce por 3 níveis de confiabilidade, caindo para o próximo apenas se o atual falhar:

L1 API / DIRECT CODE (openpyxl, Playwright) 99%
L2 UI AUTOMATION (pywinauto, Árvore de Acessibilidade) 95%
L3 VISION + PYAUTOGUI (Claude Vision GUI Fallback) 80%
🛡️ SELF-HEALING & TECH STACK

O Retry Engine diagnostica falhas e aplica 5 estratégias de recuperação (Wait, Alternative Path, Recover State, Escalate, Abort) garantindo estabilidade autônoma.
